VisGraphs.adjacency_matrix — Method
adjacency_matrix(edges, n)

Build an n×n adjacency matrix from an edge list.

Edges are treated as undirected: both A[i,j] and A[j,i] are set to 1. Works with both unweighted edges (i, j) and weighted edges (i, j, w); the weight is ignored — use the raw values if you need a weighted matrix.

Arguments: edges: edge list returned by hvg, nvg, whvg, or wnvg. n::Int: number of nodes (length of the original time series).

Returns: Matrix{Int}: symmetric n×n adjacency matrix.

Example

x = [1.0, 2.0, 3.0, 1.5]
edges = nvg(x)
A = adjacency_matrix(edges, length(x))

VisGraphs.degree_distribution — Method
degree_distribution(edges, n)

Compute the degree of every node and return the full degree sequence, plus a normalised frequency distribution.

The degree of node i is the number of edges incident to it.

Arguments: edges: edge list returned by hvg, nvg, whvg, or wnvg. n::Int: number of nodes (length of the original time series).

Returns: degrees::Vector{Int}: degree of each node (length n). distribution::Dict{Int,Float64}: maps each observed degree k to the fraction of nodes with that degree P(k).

Example

x = generate_sine(50)
edges = nvg(x)
degrees, dist = degree_distribution(edges, length(x))
println("Max degree: ", maximum(degrees))
println("P(k): ", sort(collect(dist)))

VisGraphs.laplacian_matrix — Method
laplacian_matrix(edges, n)

Compute the combinatorial Laplacian matrix L = D - A, where D is the diagonal degree matrix and A is the adjacency matrix.

The Laplacian is symmetric and positive semi-definite. Its eigenvalues encode structural properties of the graph: the number of zero eigenvalues equals the number of connected components, and the second-smallest eigenvalue (the Fiedler value) measures how well-connected the graph is.

Arguments: edges: edge list returned by hvg, nvg, whvg, or wnvg. n::Int: number of nodes (length of the original time series).

Returns: Matrix{Int}: symmetric n×n Laplacian matrix.

Example

x = generate_sine(20)
edges = nvg(x)
L = laplacian_matrix(edges, length(x))

# compute eigenvalues (requires LinearAlgebra)
using LinearAlgebra
λ = eigvals(Symmetric(float.(L)))
println("Fiedler value: ", λ[2])
