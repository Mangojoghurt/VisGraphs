# LLM Usage Statement

This project was developed primarily through manual implementation and design by the authors. The core algorithms, package architecture, and design decisions were written and reviewed by us.

## Use of AI Assistance

LLMs were used as a supportive tool during development in the following areas:

### 1. Code robustness and improvements
Helped to spot edge cases, improve function robustness, and suggest runtime optimizations.

### 2. Documentation and docstrings
Improved docstring clarity, consistency, and readability. This included rewording mathematical definitions and improving overall documentation structure.

### 3. Test generation
Assisted in generating initial drafts of unit tests. These tests were then reviewed, adjusted, and extended manually to ensure correctness and coverage.

### 4. Algorithm translation (HVG and NVG)
Helped to rewrite the implementations of the Horizontal Visibility Graph (HVG) and Natural Visibility Graph (NVG) from existing Python implementations in the `ts2vg` package for improved runtime.

## Note

All final code, algorithmic decisions, and package structure were designed, validated, and approved by the authors. LLMs were used only as a supplementary tool but no part of the development process was fully automated or delegated to AI systems without human review.
